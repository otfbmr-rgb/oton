DEMONSTRAÇÃO RÁPIDA

Passos para testar a edição de texto por duplo-clique:

1. Execute o app:

```bash
python main.py
```

2. Selecione a ferramenta `Texto` na barra de ferramentas.
3. Clique no canvas para criar um novo texto e digite algo.
4. Para editar um texto existente, dê um duplo-clique sobre ele — um diálogo será aberto para editar o texto.
5. Para preencher uma forma: selecione a ferramenta `Preencher` e clique sobre um retângulo/elipse.

Notas:
- Se quiser gravar um vídeo da demonstração, inicie uma ferramenta de gravação de tela (OBS, Xbox Game Bar, etc.) e siga os passos acima.
- Se preferir capturas públicas, tire screenshots do canvas após as ações e salve-as junto ao projeto.
